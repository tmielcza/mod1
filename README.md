# How To

```
make
./mod1 <file>
```

`<file>` is a file with the following format:
`(\(\d+,\d+,\d+\)[\s\n]*)*`

Drag and move to rotate.  
`+` and `-` to zoom in / out.  
`1`, `2`, `3`, `4`and `5` to enable respectively rain, flood, one-side flood, divine water column and siphon mode.  
`space` to stop.
